transform 操作符的实现

在得到 Flow 对象后往往我们并不会直接进行 collect 而是通过一定的操作符变换后，再进行 collect，下面以常用的 map 操作符为例，来看看操作符的实现是怎样的

可以看到，map 操作符的实现其实是利用了我们常见的操作符 transform 来实现的，其实不但 map 操作符是利用它实现的，而且 filter onEach 等操作符也是基于它来实现的，下面就是它的实现。

首先 transform 函数是 Flow 的拓展函数，所以在这个函数中可以调用到该Flow对象的 collect函数以 collect { } 的形式。在这个函数中 调用 flow 函数创建了一个新的 Flow对象，并在 flow 的 block 中调用了 上游Flow对象的 collect 函数，并对从上游收集到的每个值利用参数 transform操作进行变换。

值得注意的是，在transform操作中，如果仅仅对数据进行变换 ，那么由transform新创建的Flow（也称下游Flow）是不会有值的。所以在 map中是这样实现的emit(transform(value)
